## vite使用demo

## start

- install

```shell
npm install vite -D
```

- npm scripts
```json
{
  "scripts": {
    "build": "vite build",
    "dev": "vite"
  },
}
```

## demo-style

- install

```shell
npm install sass less -D
```

直接使用即可，无须更多配置
